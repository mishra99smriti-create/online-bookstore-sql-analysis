# online-bookstore-sql-analysis
# Online Bookstore SQL Analysis

## Project Overview
This project analyzes an online bookstore database using MySQL.  
The goal of this project is to extract business insights related to book sales, customers, orders, revenue, and inventory management using SQL queries.

---

## Tools Used
- MySQL
- SQL
- GitHub

---

## Dataset Information
The dataset contains information related to:
- Books
- Customers
- Orders
- Sales transactions
- Inventory details

---

## Key Business Questions Solved

### Sales Analysis
1. Total Revenue Generated
2. Best-Selling Books
3. Top Customers by Purchase
4. Monthly Sales Trends
5. Average Order Value

### Inventory Analysis
- Available stock analysis
- Low inventory books
- Most ordered categories

### Customer Analysis
- Customer purchase behavior
- Order frequency
- High-value customers
